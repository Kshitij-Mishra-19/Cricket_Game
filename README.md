# Cricket_Game

A lightweight **Python CLI game** inspired by the classic Hand Cricket game. Challenge the computer, score runs, and keep batting until your shot matches the computer's pick.

## ✨ Highlights
- Clean and interactive command-line interface
- Randomized computer opponent
- Real-time score tracking
- Simple, beginner-friendly Python implementation

## 🎮 Game Rules
- Choose a number between **0** and **6**.
- If your choice is different from the computer's, your runs are added to the total score.
- If both numbers match, you're **OUT** and the game ends.
